### 网上花店小程序✔Java✔SpringBoot✔Vue2✔Uniapp✔MySQL✔万字文档
## 源码 + SQL脚本 + 软件环境安装包 + 运行启动教程 + 万字文档（LW）
## 【项目采用技术】
①架构: B/S、MVC
②系统环境：Windows/Mac
③开发环境：IDEA、JDK1.8、Maven、MySQL、HBuilderX/微信开发者工具
④技术栈：Java、MySQL、SpringBoot、Mybatis/MybatisPlus、Vue2、Uniapp、ElementUI、JavaScript、CSS
## 【系统功能模块】
系统分为管理员，用户2种角色。
①小程序页面功能包括，
1、首页推荐信息，鲜花展示，公告信息展示。
2、鲜花浏览，鲜花购买，加入购物车，收藏。
3、公告信息浏览。
4、论坛交流。
5、我的（个人中心），用户充值，修改密码，收货地址，购物车，客服聊天，公告信息，鲜花，，鲜花收藏，鲜花评价，鲜花订单，论坛信息。
②管理后台功能包括，个人中心，客服聊天管理。基础数据管理，帖子类型管理，公告类型管理，鲜类型管理。论坛交流管理。公告信息管理。鲜花管理。用户管理。轮播图管理等等。
